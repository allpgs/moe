micro:bit Offline Editor

micro:bit Offline Editor는 micro:bit Makecode를 오프라인에서도 사용할 수 있게 하는 프로그램입니다.

v1.0~2.0 까지는, pxt를 기반으로 작동하였으나, v3.0 배포(21/12/15) 부터 2.0 이하 버전의 지원을 종료하고,

Microsoft사의 공식 MakeCode Offline을 다운로드 하도록 다운로더 형태로 v3.0을 배포하게 되었습니다.

설치

1.0 버전

1. 1.0 버전을 다운로드 하기 위해 아래 링크를 눌러 다운로드 해주세요.
다운로드(64비트)

2. 압축을 풀어주세요.

3. node.js를 다운로드 해주세요.

4. node.js 설치 파일을 실행해 설치를 진행하세요.

5. 아까 다운받은 MOE 설치 파일을 실행해 주세요.

6. 자동으로 설치가 진행됩니다.

2.0 버전

1. 2.0 버전을 다운로드 하기 위해 아래 링크를 눌러 다운로드 해주세요.
다운로드(32비트)
다운로드(64비트)

2. node.js를 다운로드 해주세요.

3. node.js 설치 파일을 실행해 설치를 진행하세요.

4. 아까 다운받은 MOE 설치 파일을 실행해 주세요.

5. 크롬이 없다면 설치 중 yes/no를 물을 때 yes, 있다면 no를 입력해 주세요.

6. 자동으로 설치가 진행됩니다.

3.0 버전

  1. 3.0 버전을 다운로드 하기 위해 아래 링크를 눌러 다운로드 해주세요.
    다운로드(64비트)

  2. 파일을 실행해 주세요.

  3. 라이센스에 동의 해주세요.(Microsoft 라이센스)

  4. 선택적으로 엔트리나 스크래치를 설치 할 수 있습니다. 설치하려면 선택해 주세요.

  5. 자동으로 설치가 진행됩니다.

    5-1. 선택 프로그램 체크한 경우 자동으로 설치창이 발생하고, 설치를 이어가 주세요.

  6. MakeCode가 자동 시작되면 설치가 완료됬습니다.
